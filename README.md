# 🎨 社团 HTML 年报 — 长滚动版

基于纯 CSS + JS 的 **长滚动式** 年报模板，适合内容丰富的社团年度报告。零依赖，全屏适配电脑。

## 预览方式

```
双击 start.bat
```
或
```
双击 serve.bat  →  浏览器 http://localhost:8080
```

## 内容结构

| # | 区块 | 说明 |
|:-:|------|------|
| 🎬 封面 | 社团名 + 年份 + 核心数据 | Hero 全屏 |
| 01 | 社团概览 | 核心指标卡片 |
| 02 | 社长致辞 | 长篇文字 |
| 03 | 年度视频 | 🎥 替换 MP4 |
| 04 | 活动照片 | 🖼️ 8 格照片墙，可无限加 |
| 05 | 活动回顾 | 📅 时间线，可无限加 |
| 06 | 年度数据 | 📊 数据卡片，可无限加 |
| 07 | 游戏展示 | 🎮 Unity WebGL |
| 08 | 成员寄语 | 💬 引语卡片，可无限加 |
| 09 | 未来计划 | 🚀 计划列表 |
| 🏁 封底 | 感谢 + 联系方式 | |

**所有区块都可以自由增删，滚动无限扩展。**

## 操作方式

| 方式 | 效果 |
|------|------|
| 鼠标滚轮 | 上下滚动 |
| 侧边导航（左）| 点击跳转到对应区块 |
| 顶部进度条 | 显示阅读进度 |
| 触摸滑动 | 上下滚动（手机） |

## 嵌入飞书

1. 托管到 **GitHub Pages** / **Vercel**
2. 飞书文档 → 「嵌入」→ 「嵌入网页」→ 粘贴 URL

## 技术栈

- **零依赖** — 不加载任何外部 CDN / JS 库
- Intersection Observer — 进入视口淡入动效
- CSS `clamp()` — 响应式字号
- 纯 CSS `scroll-behavior: smooth`
